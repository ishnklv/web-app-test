# Web App
### Author: Aktan Ishenkulov

## Instructions for running application
1. Install requirements use command: ``npm install``
2. Copy variables from: ``.env.example`` to `.env` use command: ``cp .env.example .env && vi .env``
3. Up database use command: ``docker-compose up -d``
4. Run migrations use command: ``npm run migration:run``
5. Run seeders use command: ``npm run seed:run``
6. Run application use command: ``npm run start``
